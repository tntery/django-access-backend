Sync Users

Syncing users will fetch the latest user data from the external accounting system and update the user list and their balances here. It will also remove any users that have been deleted from the external system. This is useful to do after making changes to users in the external system or if you want to ensure you have the latest user data.

Note: This operation may take a few moments depending on the number of users in the external system.
